# Cloud Storage Pricing Portal

 Dashboard comparing storage and egress costs across major cloud providers.

## Live Demo

[View the live dashboard](https://storage.portaljs.com/)

## Features

- Compare 17+ cloud storage providers
- Storage vs egress cost analysis
- Free egress provider highlights
- Monthly cost scenario calculator

## Data

Pricing data sourced from [provider documentation analysis](https://gist.github.com/Manouchehri/733e6235457e60de24fdbb15046fba7f).

Providers include: AWS S3, Cloudflare R2, Backblaze B2, Google Cloud Storage, Wasabi, Storj, and more.

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000
